This app uses Google Sheets as its shared database so all officers and staff see one live task list. Complete the one-time setup below — takes about 5 minutes.

One-time setup (admin only):
  1. Go to script.google.com → New Project
  2. Delete existing code → Paste the Code.gs file contents
  3. Click Deploy → New Deployment → Web App
  4. Execute as: Me | Access: Anyone
  5. Click Deploy → Authorize → copy the Web App URL
  6. Paste the URL below and click Connect
